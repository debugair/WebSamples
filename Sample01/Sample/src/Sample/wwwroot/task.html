Teilaufgabe 1

Wir schreiben unsere Client Side Logik so um, dass die Add und Edit Form sowie die Person Tabelle dynamisch generiert werden. Dazu definieren wir in unserem script.js das folgende Objekt. 

      var baseEntity = {"Id":0, "Familyname":"", "Firstname":""};

Über das Objekt baseEntity bauen wir die Dynamik auf der Client Side im JavaScript auf. Ziel sollte es sein, dass wenn wir dem Objekt einen neue Property hinzufügen sich das ganze UI um diese Property erweitert (inkl. Kommunikation mit dem REST Service).

Teilaufgabe 2

Wir testen die Dynamik aus, indem wir unserem C# Objekt eine Property Middlename anhängen. Unsere baseEntity erweitern wir genauso. 

      public class Person{
        public int Id {get;set;}
        public string Familyname {get;set;}
        public string Firstname {get;set;}
        public string Middlename {get;set;}
      }
    
      var baseEntity = {"Id":0, "Familyname":"", "Firstname":"", "Middlename":""};
