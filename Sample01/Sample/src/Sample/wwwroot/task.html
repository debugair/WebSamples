Teilaufgabe 1

Wir schreiben einen REST Service für eine Klasse Person. Diese sieht wie folgt aus: 

      public class Person{
        public int Id{get;set;}
        public string Familyname{get;set;}
        public string Firstname{get;set;}
      }

Die REST Implementation umfasst die folgenden Methoden:

Action Method Request Response
api/person GET Person[]
api/person POST Person Person
api/person PUT Person Person
api/person DELETE Id Id

Der REST Service verfügt über eine statische Repository Klasse in welcher in einer statischen Liste die Personen Objekte verwaltet werden. Die Werte werden im JSON Format hin und her geschoben (Response / Request).

Teilaufgabe 2

Im wwwroot Folder des WebApi Projektes legen wir einen Folder script und einen Folder style an. Darin legen wir die aktuelle Version von jQuery ab (jquery.min.js -> Folder script) sowie die aktuelle Version des Bootstrap CSS (bootstrap.min.css -> Folder style).

Nun legen wir zwei leere Files an, ein script.js (-> Folder script) und ein style.css (-> Folder style). im wwwroot Folder legen wir ein index.html File an in dessen head wir bootstrap und style CSS und jquery und script JS einbauen.

Nun bauen wir in der Index Seite per JavaScript die folgenden UI Elemente ein:

  • Add Person Form (mit Add Button)
  • Edit Person Form (mit Edit, Delete und Cancel Button)
  • List Person Table

Das UI muss die folgende Funktionalität bereitstellen:

  • Beim Laden der Seite die Add Form anzeigen und die Edit Form verstecken
  • Beim Laden der Seite die Person Tabelle befüllen/li>
  • Neue Person hinzufügen
  • Auf Klick einer Row in der Tabelle die Edit Form einblenden und die Add Form verstecken
  • Auf Edit Button die Änderungen in der Edit Form speichern (hide Edit / show Add)
  • Auf Delete Button die Person in der Edit Form löschen (hide Edit / show Add)
  • Auf Cancel Button hide Edit Form / show Add Form
  • Bei allen Änderungen im UI die Person Table neu laden

Auflagen sind:

  • Die Kommunikation mit dem REST Service läuft über JSON
  • Wir verwenden jQuery als JavaScript Library
  • Wir bauen als erstes den REST Service
  • Wir bauen als zweites die Client Logik
  • Wir betreiben nach erfolgreicher Implementation Refactoring
  • Wir betreiben nach erfolgreicher Implementation minimales Styling mit Bootstrap
